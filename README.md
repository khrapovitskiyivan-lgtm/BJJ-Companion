# BJJ Companion

Интерактивная карта техник бразильского джиу-джитсу: 293 техники с детальной механикой, связями (пререквизиты → продолжения), риском травмы и прогрессом обучения.

**Аудитория:** увлечённый белый пояс, жаждущий структурированной информации. RU + EN.

## Стек

- **TanStack Start** (SSR, React 19) + Tailwind 4
- **vis-network** — граф техник (иерархическая раскладка по поясам, физика отключена)
- **Supabase** — auth + синхронизация прогресса (в работе)
- Данные: `data/techniques.csv` → `scripts/build-data.mjs` → `src/lib/bjj/generated/techniques.json`

## Данные

Источник правды — `data/techniques.csv` (293 техники × 28 полей: механика, ключевые моменты, ошибки, дриллы, риск травмы, tap warning, легальность IBJJF/ADCC, связи).

После правки CSV:

```bash
npm run build:data
```

Генератор валидирует ссылки/дубли/пустые поля и падает при ошибках.

⚠️ При импорте CSV в Google Sheets отключайте «Преобразовывать текст в числа и даты» — русская локаль превращает списки id в даты.

## Разработка

```bash
npm install
npm run dev
```

## Структура контента (i18n)

Контент техник хранится по локалям: `Technique.content.ru` (полный), `content.en` (этап перевода). Имена уже двуязычные (`nameRu`/`nameEn`). UI-строки — `src/lib/i18n.ts`.

## Монетизация (план)

- **Free:** вся карта, все текстовые карточки, прогресс, поиск, фильтры
- **Paid:** авторские видео к техникам, программы обучения, «что делать если», интервальные повторения
