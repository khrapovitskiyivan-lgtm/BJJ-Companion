# Ревизия базы на отсутствующие техники и переходы

Дата: 2026-07-22. Роль: `.claude/agents/bjj-gap-auditor.md` (аудит контролёром в роли).
Повод: отзывы из зала (нет кросс-чоука из халф-гарда, нет перехода Дип халф → X-гард).
Метод: инвентарь 310 карточек по группам + позиционная матрица (входы/атаки/побеги/
переходы) + проверка связности графа. Правки существующих формулировок — не здесь.

## Главный вывод
База **плотная и почти полная** (310 карточек: 58 позиций, 78 сабмишенов, 34 свипа,
29 проходов, 31 переход, 27 побегов, 29 тейкдаунов). Несколько «пробелов» из отзывов
и первичных гипотез при проверке оказались УЖЕ покрыты под другими названиями:
- Сайд → Колено на животе = «Переход колено на корпусе» (354, из сайда). Есть.
- Paper cutter / бумагорез = «Режущее удушение» (201). Есть.
- Кимура из халфа = покрыта системой ловушек кимуры (368) + Кимура-трэп (502).

Поэтому список реальных дыр КОРОТКИЙ. Это хорошо: не надо раздувать базу ради галочки.

## Переходы-дыры (симметрия графа)
- Дип халф (12) не ведёт в X-гард (10), хотя обратно связь заявлена. Чинится карточкой №2 + `12.chain_to += 662`, `10.setup_from += 662`.
- Маунт (16) не ведёт в S-маунт (649). Чинится карточкой №3.
- Мелкие асимметрии (халф→дип халф) — по желанию (№5).

## Что НЕ является пробелом (проверено, есть)
Армбары/треугольники/кимуры/омоплаты/американы по всем позициям; полный набор ги/ноги
удушений (RNC, кросс-коллар, иезекиль, гильотины, дарс/анаконда, лук-стрела, петля,
брабо, часовое/режущее, бейсбольная бита, север-юг, фон флю, солья, де пассада,
гифт рап, багги, бульдог, распятие, японский/перуанский галстук); леглок-система
(прямой ахилл, хилхук ин/аут, тохолд ин/аут, нибар/реверс, эстима, слайсеры);
проходы (нож/коленом, дабл/овер-андер, тореадо, лег-драг, лонг степ, смэш, X-пасс,
стэкинг, бэкстеп, боди-лок, Сайборг, Ксиси); свипы; побеги основных сабмишенов;
тейкдауны (сингл/дабл/лоу сингл, хай кротч, энкл пик, арм-драг, рашн-тай, снэп-даун,
дак-андер, спрол + броски дзюдо). Ретеншн-набор. 11 систем-обзоров.
