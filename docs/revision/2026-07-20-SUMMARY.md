## Замечания вне зоны ревизии (не правил — на ваше решение)
- **em-dash по всей базе**: длинное тире «—» используется во всех 293 техниках, вопреки правилу проекта «без em-dash». Массовая правда, лечится одним отдельным проходом (не смешивал с ревизией).
- **Легальность бинарными флагами**: id 500 (heel hook) помечен `legal_ibjjf_gi=✔`, хотя запрещён; id 362 (bicep slicer), 359 (twister), 383 (ристлок) — флаги не отражают поясные/форматные ограничения (есть только да/нет).
- **id 17 (контроль спины)**: `points_adcc=2`, но со скрещёнными крюками ADCC даёт 3.
- **Плейсхолдеры связей**: у ряда проходов гарда `common_setups = 130, 37` (не сетапы прохода); в takedown `common_setups` полностью дублирует `setup_from`. Кандидаты на отдельную чистку.
- **«Де ла Рива» vs «Де-ла-Рива»**: глоссарий без дефисов, база с дефисами — выровнять в одну сторону.
- `success_rate` — оценочные проценты без источника; возможно, стоит пометить в UI как условные.
